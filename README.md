# Azure VM Static Website Hosting

## Overview

This project demonstrates how to create an Ubuntu Virtual Machine on Microsoft Azure and host a static website using Apache HTTP Server.

## Features

- Azure Virtual Machine
- Ubuntu 22.04 LTS
- Apache Web Server
- Static Website Hosting
- SSH Remote Access

## Technologies Used

- Microsoft Azure
- Ubuntu Linux
- Apache2
- HTML
- CSS
